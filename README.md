## 📌 프로젝트 소개

FlowOps는 운영팀이 사용하는 요청 관리 B2B SaaS 서비스의 포트폴리오 프로젝트입니다.
메신저, 이메일, 스프레드시트에 흩어진 요청을 접수부터 배정, 처리, 리포트까지 한 곳에서 관리할 수 있는 운영 대시보드입니다.

## 🖥️ 주요 화면

| 화면 | 설명 |
|------|------|
| 01 로그인 | 서비스 소개 히어로 + 로그인 폼 |
| 02 대시보드 | KPI 카드, 상태 분포 차트, 최근 요청 |
| 03 요청 관리 | 필터, 검색, 일괄 처리 테이블 |
| 04 요청 상세 | 기본 정보, 댓글, 활동 이력 |
| 05 새 요청 | 표준화된 요청 접수 폼 |
| 06 멤버 관리 | 역할 기반 권한 구조 |
